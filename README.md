# 🏔️ 黄山冬季两日游互动指南

一个专为黄山冬季旅行设计的交互式网页指南，包含详细的行程规划、装备清单、预算分析和安全提示。

## ✨ 功能特点

- 📅 **交互式行程规划**：两日行程详细时间表，支持切换查看
- 📊 **海拔变化图表**：可视化展示徒步路线的海拔变化，帮助合理分配体力
- 🎒 **智能装箱清单**：可点击标记的装备清单，确保不遗漏重要物品
- 💰 **预算分析**：使用饼图展示费用构成，提供省钱建议
- ⚠️ **安全提示**：冬季黄山重要注意事项和关闭区域提醒
- 🚄 **交通信息**：推荐的高铁车次和时间表

## 🛠️ 技术栈

- HTML5
- Tailwind CSS (CDN)
- Chart.js (用于图表展示)
- 纯 JavaScript (无框架依赖)

## 📝 使用说明

1. 打开网页后，可以通过顶部导航切换不同板块
2. 在"行程规划"页面，可以切换查看 Day 1 和 Day 2 的详细行程
3. 在"装备与预算"页面，点击装备清单中的项目可以标记为已打包
4. 所有数据都是静态的，无需后端服务器

## ⚠️ 重要提示

- 本指南基于计划日期生成，出行前请务必核对：
  - 官方天气预报
  - 索道运营时间
  - 景区开放情况
- 冬季黄山路滑，务必携带冰爪等防滑装备
- 注意索道停运时间，避免错过下山
